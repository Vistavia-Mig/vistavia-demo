# Vistavia — Next.js Starter (App Router + Tailwind)

**Qué es:** un proyecto mínimo de Next.js con Tailwind y el **demo Vistavia** listo para publicar en **Vercel**.

## Cómo publicarlo sin instalar nada
1) Crea una cuenta en GitHub y en Vercel (gratis).
2) En GitHub, crea un repositorio nuevo vacío (p.ej., `vistavia-demo`).
3) Sube los archivos de este ZIP al repo (Add file → Upload files).
4) En Vercel: New Project → Import Git Repository → elige tu repo → Deploy.

## Edita primero
- En `app/page.tsx` cambia `WHATSAPP_LINK` y `YAPE_LINK`.
- El favicon ya está en `public/favicon.svg` (isotipo Vistavia).
